# ⚖️ Online Legal Consultation Platform

A full-stack web platform that connects users with verified lawyers for online legal consultation.

## 🚀 Features

* User & Lawyer Authentication
* Lawyer Discovery & Filtering
* Appointment Booking System
* Admin Dashboard

## 🛠️ Tech Stack

* Frontend: Next.js, Tailwind CSS
* Backend: Node.js, Express.js
* Database: MongoDB

## 📦 Setup Instructions

```bash
git clone <repo-link>
cd legal-platform
```

### Start Backend

```bash
cd server
npm install
npm run dev
```

### Start Frontend

```bash
cd client
npm install
npm run dev
```
