# ⚖️ Online Legal Consultation Platform

A full-stack web application that connects users with verified lawyers for online consultation, appointment booking, and video calls.

---

## 🚀 Features

### 👤 Authentication

* User & Lawyer registration/login (JWT-based)
* Role-based access control

### ⚖️ Lawyer Marketplace

* Browse lawyers by specialization, location, experience, fees
* Search & filter functionality

### 📅 Appointment System

* Book appointments with lawyers
* Status tracking (pending, accepted, rejected)

### 🎥 Video Consultation

* Real-time video calls using ZegoCloud
* Appointment-based meeting rooms

### ⭐ Reviews & Ratings

* Users can rate and review lawyers
* Average rating displayed

### 📊 Dashboard

* User dashboard (appointments)
* Lawyer dashboard (earnings, requests)
* Admin dashboard (analytics, approvals)

---

## 🛠️ Tech Stack

### Frontend

* Next.js (App Router)
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB

### Other

* JWT Authentication
* ZegoCloud (Video Calling)

---

## 📦 Installation

```bash
git clone <your-repo-link>
cd legal-platform
```

### Backend Setup

```bash
cd server
npm install
npm run dev
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```
